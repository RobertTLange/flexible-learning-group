# 'Flexible Learning' Reading Group @ TU Berlin

In this repository the different papers and presentations discussed during the "Flexible Learning" Reading Group are documented. Topics range from Deep Learning (Meta, Transfer, Continual Learning) to Systems Neuroscience (Consolidation, Plasticity, Learning).
